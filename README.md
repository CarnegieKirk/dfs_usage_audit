# DFS Auditing Tool

## Usage
```
target/aarch64-apple-darwin/release/dfs_usage_audit --help
Usage: dfs_usage_audit [OPTIONS] --path <PATH>

Options:
  -p, --path <PATH>
  -t, --threads <THREADS>  [default: 50]
  -d, --days <DAYS>        [default: 1095]
  -h, --help               Print help
  -V, --version            Print version
```
